# Network Lead Pro - Production SaaS CRM

Aplikasi manajemen jaringan, otomatisasi follow up prospek, dan pengumpulan kampanye penyiaran berbasis multi-tenant yang aman dan real-time.

## Teknologi Utama
- **Frontend**: React 18, Vite, TypeScript, Material UI (MUI v5), Zustand
- **Backend & Database**: Supabase (PostgreSQL, Row Level Security, Realtime Sync)
